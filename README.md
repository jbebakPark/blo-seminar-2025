# 🔧 Gallery.html 수정 완료

## ✅ 수정 내용

### 1. QR 코드 표시 문제 해결
**문제**: QR 코드 이미지가 보이지 않음  
**원인**: QR 코드가 외부 파일 참조 방식으로 되어 있었음  
**해결**: QR 코드를 Base64로 인코딩하여 HTML 파일에 직접 임베드

**변경 전**:
```javascript
document.getElementById('qrCode').src = 'frame.png';
```

**변경 후**:
```javascript
document.getElementById('qrCode').src = 'data:image/webp;base64,UklGRvi2AABXRUJQVl...';
```

### 2. 텍스트 변경
**변경 내용**: '삼성글로벌캠퍼스' → '삼성금융캠퍼스'

**변경된 위치** (총 3곳):
1. 오프라인 참석 안내 - 장소 소개
2. 오프라인 참석 정보 - 장소 상세

## 🐛 문제 해결

### Q: QR 코드가 여전히 표시되지 않는다면?
**A**: 브라우저 캐시를 완전히 삭제하세요:
- Chrome: Ctrl + Shift + Delete → 모두 삭제
- Safari: 개발 → 캐시 비우기
- 또는 시크릿/프라이빗 모드에서 테스트

### Q: 파일 크기가 너무 크다면?
**A**: 현재 1.8MB는 웹 페이지로서 적절한 크기입니다. 필요시 다음 방법으로 최적화:
- 기타 이미지들을 WebP로 변환
- 불필요한 Base64 데이터 제거
- 이미지 압축

### Q: QR 코드 스캔이 안 된다면?
**A**: 
1. QR 코드 이미지가 손상되지 않았는지 확인
2. 밝은 곳에서 스캔 시도
3. 다른 QR 코드 리더 앱 사용
